# FastAPI Docker App

A simple FastAPI application with Docker and Docker Compose.

## Description

This project demonstrates how to containerize a FastAPI application using Docker and run it with Docker Compose.

## Features

- FastAPI web framework
- Docker containerization
- Docker Compose for easy deployment

## Getting Started

1. Clone the repository
2. Build the Docker image: `docker build -t fastapi-docker-app:latest .`
3. Run with Docker Compose: `docker-compose up -d`
